# clm-ai-anomaly
Backend-as-a-service pour gérer des contrats (PDF). Extraction automatique (LLM), recherche sémantique vectorielle, détection d'anomalies financières (statistiques). Architecture asynchrone (FastAPI, Celery, Redis), conteneurisée (Docker). Projet pédagogique illustrant la POO avancée (descripteurs, protocoles, mixins).
